## Option Stores

–ü–æ–¥–æ–±–Ω–æ Vue's Options API, –º—ã —Ç–∞–∫–∂–µ –º–æ–∂–µ–º –ø–µ—Ä–µ–¥–∞—Ç—å –æ–±—ä–µ–∫—Ç Options —Å–æ —Å–≤–æ–π—Å—Ç–≤–∞–º–∏ `state`, `actions` –∏ `getters`.

```js {2-10}

    getters: {
        doubleCount: (state) => state.count * 2,

    actions: {
        increment() {
            this.count++
        },
    },
})
```
